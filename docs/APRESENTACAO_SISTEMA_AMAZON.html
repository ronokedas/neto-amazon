Sistema Amazon
04
Diferenciais técnicos

Base técnica pensada para confiança, rastreabilidade e implantação real.

Os diferenciais técnicos não aparecem como detalhe: eles reduzem risco para o comprador e aumentam a capacidade de operar documentos sensíveis com segurança.

Assinatura digital

Token único criptográfico, rota pública de assinatura, registro de IP, data, hora e imagem da assinatura para propostas e certificados.

PDF com hash

Propostas, relatórios e certificados oficiais são gerados em PDF com hash SHA-256 e persistência segura.

Storage MinIO / S3

Arquivos, comprovantes, imagens e documentos podem ser armazenados em S3 ou MinIO, com fallback para armazenamento local.

Segurança aplicada

CSRF, validação de CPF/CNPJ, senhas com hash, sessão por inatividade, permissões por cargo, logs e soft delete.

Ambiente Docker

PHP 8.2, MySQL 8.0 e MinIO em ambiente conteinerizado, com deploy automatizado para Ubuntu/Debian.

Numeração sem duplicidade

Geração atômica com SELECT FOR UPDATE para documentos como AM-CSN, AM-CNBL, AM-CNARQ e AM-REL-HT.

Tecnologias utilizadas: PHP 8.2, MySQL 8.0, Docker, MinIO/S3, PHPMailer, JavaScript, CSS3, HTML5, AWS SDK e PDO.
